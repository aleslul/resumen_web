# Estudia FAB

Un proyecto web para crear y compartir recursos educativos, con un enfoque en diseño limpio y experiencia de usuario accesible.

## Características

- Estructura de proyecto moderna para aplicaciones web.
- Interfaz responsiva para dispositivos móviles y escritorio.
- Enfoque en facilidad de uso y gestión de contenido.

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone <url-del-repositorio>
   cd estudia-fab
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```

## Uso

Iniciar el servidor de desarrollo:
```bash
npm start
```

Abrir el navegador en `http://localhost:3000` o la dirección especificada en la terminal.

## Estructura del proyecto

- `src/` - Código fuente de la aplicación.
- `public/` - Archivos estáticos.
- `README.md` - Documentación del proyecto.
